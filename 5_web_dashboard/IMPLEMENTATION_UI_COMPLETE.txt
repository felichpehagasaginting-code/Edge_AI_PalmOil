📡 API ENDPOINTS (NEW)
═══════════════════════════════════════════════════════════════════════════════

GET /api/monitoring/errors/recent
  └─ Query params: hours, limit, component, severity
  └─ Response: { count, errors[], filters, timestamp }

GET /api/monitoring/errors/stats
  └─ Query params: minutes
  └─ Response: { statistics, total_errors, critical_count, error_rate_per_hour }

GET /api/monitoring/errors/by-component
  └─ Query params: hours
  └─ Response: { components{}, total_components }

GET /api/monitoring/alerts/active
  └─ Query params: limit
  └─ Response: { count, alerts[] }

GET /api/monitoring/components/health
  └─ Response: { components{}, overall_status, critical_count, degraded_count }

GET /api/monitoring/performance/metrics
  └─ Query params: minutes
  └─ Response: { operation_stats, database_query, api_response }

GET /api/monitoring/system/overview
  └─ Response: { summary, errors, performance, components }

POST /api/monitoring/alerts/{id}/acknowledge
  └─ Response: { status: "success", alert_id }


🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Add Monitoring API to API Server
  1. Copy monitoring_api.py ke backend folder
  2. Dalam api_server.py, import:
     from monitoring_api import router as monitoring_router
     from monitoring_api import set_monitoring_services

  3. Initialize monitoring services dalam lifespan:
     error_tracker = ErrorTracker(logger=logger)
     perf_monitor = PerformanceMonitor(logger=logger)
     db_logger = DatabaseLogger(pool, logger=logger)
     
     set_monitoring_services(error_tracker, perf_monitor, db_logger, logger)
     app.include_router(monitoring_router)

STEP 2: Copy Dashboard Files
  1. Copy monitoring.html ke frontend folder
  2. Copy monitoring.js ke frontend/js folder

STEP 3: Update Nginx (jika menggunakan)
  location /monitoring {
      alias /var/www/html/monitoring.html;
  }

STEP 4: Access Dashboard
  Open browser: http://localhost:8080/monitoring

STEP 5: Test It
  - View Overview tab (statistics & charts)
  - Check Errors tab (filterable error table)
  - Review Alerts tab (active alerts)
  - Inspect Components tab (health status)
  - Analyze Performance tab (metrics & charts)
